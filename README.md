# Wan-Ho Hsu | Personal Web Page

A premium, minimalist single-page personal website for **Wan-Ho Hsu**, designed with a clean and elegant aesthetic inspired by modern luxury templates like "Kaleo".

## ✨ Features
- **Dynamic Live Clock**: A real-time digital clock and date display that updates every second.
- **Modern Typography**: High-contrast serif headings (**EB Garamond**) paired with elegant sans-serif body text (**Inter**).
- **Premium Background**: An atmospheric, serene landscape background for a professional and peaceful first impression.
- **Interactive UX**: Custom cursor-follower effect and smooth scroll-reveal animations.
- **Fully Responsive**: Optimized for all screen sizes (Desktop, Tablet, Mobile).

## 📂 Project Structure
- `index.html`: Main structure and content using semantic HTML5.
- `style.css`: Custom CSS with a warm beige/charcoal color palette and advanced layout transitions.
- `script.js`: Interactive logic for the live clock, custom cursor, and intersection observer animations.
- `assets/`: Directory containing high-quality visual assets.

## 🚀 Getting Started
1. Clone this repository:
   ```bash
   git clone https://github.com/hsu24/DIC1.git
   ```
2. Open `index.html` in any modern web browser to view the site.
